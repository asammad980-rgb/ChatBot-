## Requirements

Before running the project, ensure the following resources are available.

### 1. Diabetes Dataset

Download the diabetes dataset and place it in the project directory.

Example:

```
Diabetes_prediction.csv
```

---

### 2. Chest X-ray Dataset

The chest X-ray dataset should be organized as shown below:

```
chest_xray/
│
├── train/
│   ├── NORMAL/
│   │   ├── image1.jpeg
│   │   ├── image2.jpeg
│   │   └── ...
│   │
│   └── PNEUMONIA/
│       ├── image1.jpeg
│       ├── image2.jpeg
│       └── ...
│
└── test/
    ├── NORMAL/
    │   ├── image1.jpeg
    │   └── ...
    │
    └── PNEUMONIA/
        ├── image1.jpeg
        └── ...
```

---

### 3. Gemini API Key

Generate a Gemini API Key from Google AI Studio and configure it before running the chatbot.

---

### 4. Image Path for Prediction

Update the image path in the prediction code before testing.

Example:

```python
image_path = "/content/chest_xray/test/NORMAL/NORMAL2-IM-0290-0001.jpeg"
```

or

```python
image_path = "/content/chest_xray/test/PNEUMONIA/person1_bacteria_1.jpeg"
```

Replace the path with the image you want to test.

---

### 5. Required Python Libraries

Install the required libraries before running the project.

```bash
pip install numpy pandas matplotlib scikit-learn tensorflow pillow google-generativeai
```
